# Portafolio · Leandro Cruces

Portafolio personal de Leandro Cruces, desarrollador full-stack. Construido con
Next.js 16, Tailwind CSS 4 y framer-motion, con una estética oscura premium y
acento ámbar.

## Secciones

- **Hero** con fondo shader animado (Three.js), título con animaciones por línea
  y subtítulo tipo máquina de escribir.
- **Proyectos** en una línea de tiempo interactiva con las webs desplegadas en
  producción (Beylhe, As Rey Dry Gin, Esteko, Enbox, Crecer).
- **Musicdy** — la startup de música: explorador interactivo de capacidades con
  capturas reales de la app y una pila de imágenes ampliable (lightbox).
- **Contacto** con enlaces a redes y correo.

## Desarrollo

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # build de producción
npm run start    # servir el build
```

## Stack

Next.js · React 19 · Tailwind CSS 4 · framer-motion · lucide-react

## Deploy

Pensado para desplegarse en [Vercel](https://vercel.com): importar este
repositorio y desplegar sin configuración extra.
